# 🚀 Credit Card Fraud Detection (CCFD)

## 📌 Project Overview
This project aims to detect fraudulent credit card transactions using machine learning techniques. It helps financial institutions identify suspicious activities and prevent financial losses by analyzing transaction patterns in real-time.

---

## 🎯 Objectives
- Detect fraudulent transactions with high accuracy  
- Minimize false positives  
- Build a reliable and scalable ML model  
- Automate fraud detection using data-driven approaches  

---

## 📊 Dataset Information
The dataset contains anonymized credit card transaction records with the following features:

- `Time` – Time elapsed between transactions  
- `Amount` – Transaction amount etc.
- `Class` – Target variable  
  - `0` → Legitimate  
  - `1` → Fraudulent  

---

## 🔍 Methodology

### 1. Data Preprocessing
- Handled missing values (if any)  
- Scaled features using normalization/standardization  
- Managed imbalanced data using resampling techniques  

### 2. Exploratory Data Analysis (EDA)
- Visualized fraud vs non-fraud transactions  
- Identified patterns and correlations  

### 3. Model Building
- Logistic Regression  
- Decision Tree  
- Random Forest  

### 4. Model Evaluation
- Accuracy Score  
- Precision  
- Recall  
- F1-score  
- Confusion Matrix  

---

## 📈 Results
- Achieved high accuracy in fraud detection  
- Improved recall to capture maximum fraudulent cases  
- Reduced false positives effectively
